## Getting Started

安裝教學  
### Installation

前往[這裡](https://github.com/yoni13/HenLin_CloudQuiz_Hack/releases)下載最新版本（記得選擇Source code
(zip)）  
前往chrome://extensions或edge://extensions  
開啟右上角的開發者模式  
解壓縮剛剛下載的zip檔  
左上角載入  
重新載入翰林的網頁  
等個幾秒
